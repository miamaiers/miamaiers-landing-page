# miamaiers-personal-landing-page

A simple static HTML personal landing page (BAIS3300 Module 8 homework).

## Project Structure

- `index.html` — The single-page HTML site

## Running

Served via Python's built-in HTTP server on port 5000.

**Workflow:** `Start application` — `python3 -m http.server 5000 --bind 0.0.0.0`

## Deployment

Configured as a **static** site with `publicDir: "."`.
